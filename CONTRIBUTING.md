# Contributing to AKSHAR

Thanks for your interest in AKSHAR.

This is currently a solo indie project 
built and maintained by one developer.

## How you can help right now

- Play the game and report bugs
- Open an issue if something feels broken
- Suggest balance changes for abilities
- Report unclear UX moments

## Reporting bugs

Open a GitHub issue with:
- What you expected to happen
- What actually happened
- Browser and device you were on
- Room code if it was a multiplayer bug

## Feature suggestions

Open an issue tagged [suggestion].
No guarantee it gets built but 
everything gets read.

## Code contributions

Not open for contributions yet.
Will update this when that changes.
