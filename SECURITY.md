# Security Policy

## Reporting a vulnerability

If you find a security vulnerability in AKSHAR 
please do not open a public GitHub issue.

Email directly: prasannapandharikar19@gmail.com

Include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact

I will respond within 72 hours.

## Scope

- akshar19.vercel.app
- Firebase Realtime Database rules
- Authentication flow

## Out of scope

- Typing speed manipulation (not a competitive 
  integrity issue at current scale)
- Bot lobbies in solo mode by design
